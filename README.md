# 🎮 Tic Tac Toe AI

A modern, interactive **Tic Tac Toe game** built with **HTML, CSS, and JavaScript**, featuring both **2-player mode** and **AI gameplay with multiple difficulty levels**.

---

## 🚀 Live Demo

🔗 https://irinemicheal.github.io/tic-tac-toe/

---

## ✨ Features

* 🎯 **Two Game Modes**

  * Player vs Player
  * Player vs AI

* 🧠 **AI Difficulty Levels**

  * Easy (Random moves)
  * Hard (Minimax Algorithm)

* 👥 **Player Customization**

  * Enter player names before starting

* 📊 **Scoreboard**

  * Tracks Player 1, Player 2/AI, and Draws

* 🎨 **Modern UI**

  * Clean design with responsive layout

* 🔄 **Restart & Menu Options**

  * Restart game anytime
  * Return to menu easily

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Logic:** Minimax Algorithm (AI)

## ⚙️ How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/irinemicheal/tic-tac-toe.git
```

2. Open the folder:

```bash
cd tic-tac-toe
```

3. Run the app:

* Open `index.html` in your browser

---

## 🌐 Deployment

This project is deployed using **GitHub Pages**.

To deploy your own:

1. Push code to GitHub
2. Go to **Settings → Pages**
3. Select:

   * Branch: `main`
   * Folder: `/ (root)`
4. Save and wait for deployment

---

## 🧠 AI Logic

The AI uses the **Minimax algorithm**, which:

* Evaluates all possible moves
* Chooses the optimal move
* Makes the AI unbeatable in hard mode
